# Folder layout

- `interop`: Contains structures that interop with host and require uploads from host
    - e.g. camera parameters, which needs to be passed from C++ to Slang side

- `inter-stage`: Contains structures that interops between different rendering stages
    - e.g. auto-exposure results, which are generated by auto-exposure stage and used by composite stage

- `internal`: Shader modules that are for internal usage within the same rendering stage (needed by C++ code for creating buffers etc. though)
    - e.g. auto-exposure stage constants, shared by all shaders within auto-exposure stage

- `model`: Contains structures for model data
