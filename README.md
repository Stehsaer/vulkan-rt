# Vulkan RT

Personal experimental vulkan ray-tracing project. Currently WIP.

## Features

- Indirect rendering with GPU frustum culling
- Deferred rendering, PBR supported
- Auto exposure

## Build Prerequisites

- Xmake build system ([Link](https://xmake.io/guide/quick-start.html#installation))
- C++23 compatible compiler. Versions known to be working:
    - GCC 15+
- Slang Compiler 2026.1+ ([Link](https://github.com/shader-slang/slang/releases))

## Dev Prerequisites

In addition to build prerequisites:

- Clang-format tool 22+
